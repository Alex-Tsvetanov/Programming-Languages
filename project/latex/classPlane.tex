\doxysubsection{Detailed Description}
Represents an aircraft with details for efficient airline management. 

The \mbox{\hyperlink{classPlane}{Plane}} class encapsulates data about the manufacturer, model, seating capacity, minimum runway length, operational costs, fuel consumption, tank volume, and average speed.
